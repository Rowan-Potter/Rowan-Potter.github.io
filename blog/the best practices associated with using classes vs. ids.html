What are the best practices with using classes vs. ids?


Firstly what is a class and what is an id and why are they used? Classes and ids are basically containers,
you use said containers to organize and group your code, these containers are mainly used for styling purposes,
an ID is a more specific container used for a single line of code, think of it like a personalized name or label,
while a class is far broader, a class can be applied to as many items as you want, its in the name yet again,
a class is like this cohort it contains many different people . Its worth noting that is you apply
a class to an item with an id and they conflict the id will overide the class not vice versa.
